# Move37 Dashboard

A sleek, minimal dashboard built with Next.js and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
